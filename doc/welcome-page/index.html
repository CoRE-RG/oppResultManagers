Getting started - OMNeT++ Result Managers

You successfully installed the OMNeT++ Result Managers tools provided by the CoRE research group. On this page we would like to give you some tips on how to get started.

Requirements

Depending on the features you enabled, OMNeT++ Result Managers require further software to be installed on your computer:

  • PostgreSQL Recoder requires libpqxx
  • SQLite Recoder requires libsqlite3

Documentation

  • A changelog can be found in the WHATSNEW file
  • More info about the installation procedure can be found in the INSTALL file
  • The documentation can be found online, or generated using OMNeT++ by selecting the CoRE4INET project and then opening "Project" -> "Generate NED Documentation..." (README). After generating you find the documentation in the doc folder

Examples

  • You find simulation examples in the "examples" folder

Help

  • If you need help, you can contact us through the download page, or our webpage
  • You can find the bugtracker on the download page
